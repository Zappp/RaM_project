# RaM-app

Rick & Morty app with React, Supabase, and GraphQL.

## Setup

1. Copy environment variables:
   ```bash
   cp .env.example .env
   ```

2. Fill in your Supabase credentials in `.env`

3. Install dependencies:
   ```bash
   npm run install:api
   ```

## Run

```bash
# API
npm run dev:api
```
