# RaM-app

Rick & Morty app built with React, Supabase, and GraphQL.

## Tech Stack

- **Frontend**: Next.js 14 (App Router) + TypeScript + React Query
- **Backend**: Supabase Edge Functions (Deno) + Hono + GraphQL Yoga
- **Database**: Supabase (PostgreSQL)
- **GraphQL API**: Rick & Morty API (https://rickandmortyapi.com)
- **Testing**: Vitest (Frontend), Deno.test (Backend), Playwright (E2E)
- **Documentation**: Magidoc

## Project Structure

```
RaM-app/
├── frontend/          # Next.js application
├── api/               # Supabase Edge Functions
├── codegen/           # GraphQL Code Generator config
├── docs/              # Generated API documentation
└── e2e/               # E2E tests (Playwright)
```
